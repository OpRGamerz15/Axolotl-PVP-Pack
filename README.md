# 🪸 Axolotl PVP+

Axolotl PVP+ is a lightweight, high-performance Minecraft resource pack built purely for competitive PvP.
It focuses on smooth gameplay, clean visuals, and maximum FPS so you can dominate every fight.

---

## ⚔️ Features

* 🚀 Optimized for high FPS (low-end friendly)
* 🎯 Clean and minimal PvP textures
* 🛡️ Reduced visual clutter for better focus
* 💥 Smooth combat experience
* 🎮 Designed for competitive gameplay

---

## 📦 Installation

1. Download the latest version of **Axolotl PVP+**
2. Open Minecraft
3. Go to **Options → Resource Packs**
4. Click **Open Pack Folder**
5. Move the `.zip` file into the folder
6. Select the pack in-game

---

## 🧩 Compatibility

* Works with vanilla Minecraft
* No mods required
* Supports most PvP servers
